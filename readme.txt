Description: This is a roguelike based off binding of isaac where the goal is to reach the last floor 
while clearing rooms with mosnter in them. 

How to run: All you need to do is have all the files in the same folder and run the file
named binding of isaac. 

Libraries: random and math

Controls: wasd to move, arrow keys to shoot, press space to start the game. Some shorcut keys are
c which sets the room to clear allowing you to go through doors without clearing the room and 
b which teleports you to the boss room of the floor. h to toggle help screen. 
1, 2, 3, 4, 5, to restart the game with different number of floors. r to restart.
